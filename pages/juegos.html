Juego Interactivo

Este juego propone al usuario interactuar con criaturas mecánicas, poniendo a prueba la memoria y la observación mediante un sistema de emparejamiento.

El desarrollo del juego se realizó utilizando JavaScript con el apoyo de herramientas de inteligencia artificial para la estructura inicial y la resolución de problemas durante el proceso.
